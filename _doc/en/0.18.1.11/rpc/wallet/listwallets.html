--- layout: doc name: listwallets elemversion: 0.18.1.11 elemgroup: wallet permalink: /en/doc/0.18.1.11/rpc/wallet/listwallets/ --- listwallets Returns a list of currently loaded wallets. For full information on the wallet, use "getwalletinfo" Result: [ (json array of strings) "walletname" (string) the wallet name ... ] Examples: > elements-cli listwallets > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listwallets", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:7041/
