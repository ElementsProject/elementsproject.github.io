--- layout: doc name: dumpblindingkey elemversion: 22.0.0 elemgroup: wallet permalink: /en/doc/22.0.0/rpc/wallet/dumpblindingkey/ --- dumpblindingkey "address" Dumps the private blinding key for a CT address in hex. Note: If the address is not a CT address, looks for blinding key corresponding to this non-CT address. Arguments: 1. address (string, required) The CT address Result: "str" (string) the blinding key Examples: > elements-cli dumpblindingkey "my address"
