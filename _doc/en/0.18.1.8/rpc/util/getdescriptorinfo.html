Arguments: 1. descriptor (string, required) The descriptor. Result: { "descriptor" : "desc", (string) The descriptor in canonical form, without private keys "isrange" : true|false, (boolean) Whether the descriptor is ranged "issolvable" : true|false, (boolean) Whether the descriptor is solvable "hasprivatekeys" : true|false, (boolean) Whether the input descriptor contained at least one private key } Examples: Analyse a descriptor > elements-cli getdescriptorinfo "wpkh([d34db33f/84h/0h/0h]0279be667ef9dcbbac55a06295Ce870b07029Bfcdb2dce28d959f2815b16f81798)"
