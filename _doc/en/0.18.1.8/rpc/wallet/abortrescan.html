--- layout: doc name: abortrescan elemversion: 0.18.1.8 elemgroup: wallet permalink: /en/doc/0.18.1.8/rpc/wallet/abortrescan/ --- abortrescan Stops current wallet rescan triggered by an RPC call, e.g. by an importprivkey call. Examples: Import a private key > elements-cli importprivkey "mykey" Abort the running wallet rescan > elements-cli abortrescan As a JSON-RPC call > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "abortrescan", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:7041/
