--- layout: doc name: joinpsbts elemversion: 0.18.1.8 elemgroup: rawtransactions permalink: /en/doc/0.18.1.8/rpc/rawtransactions/joinpsbts/ --- joinpsbts ["psbt",...] Joins multiple distinct PSBTs with different inputs and outputs into one PSBT with inputs and outputs from all of the PSBTs No input in any of the PSBTs can be in more than one of the PSBTs. Arguments: 1. txs (json array, required) A json array of base64 strings of partially signed transactions [ "psbt", (string, required) A base64 string of a PSBT ... ] Result: "psbt" (string) The base64-encoded partially signed transaction Examples: > elements-cli joinpsbts "psbt"
