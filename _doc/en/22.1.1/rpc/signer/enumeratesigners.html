--- layout: doc name: enumeratesigners elemversion: 22.1.1 elemgroup: signer permalink: /en/doc/22.1.1/rpc/signer/enumeratesigners/ --- enumeratesigners Returns a list of external signers from -signer. Result: { (json object) "signers" : [ (json array) "hex", (string) Master key fingerprint "str", (string) Device name ... ] } Examples: > elements-cli enumeratesigners > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "enumeratesigners", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:7041/
