--- layout: doc name: uptime elemversion: 22.1.1 elemgroup: control permalink: /en/doc/22.1.1/rpc/control/uptime/ --- uptime Returns the total uptime of the server. Result: n (numeric) The number of seconds that the server has been running Examples: > elements-cli uptime > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "uptime", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:7041/
