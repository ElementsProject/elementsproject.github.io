--- layout: doc name: savemempool elemversion: 23.2.1 elemgroup: blockchain permalink: /en/doc/23.2.1/rpc/blockchain/savemempool/ --- savemempool Dumps the mempool to disk. It will fail until the previous dump is fully loaded. Result: { (json object) "filename" : "str" (string) the directory and file where the mempool was saved } Examples: > elements-cli savemempool > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "savemempool", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:7041/
