--- layout: doc name: combinepsbt elemversion: 0.21.0.0 elemgroup: rawtransactions permalink: /en/doc/0.21.0.0/rpc/rawtransactions/combinepsbt/ --- combinepsbt ["psbt",...] Combine multiple partially signed Bitcoin transactions into one transaction. Implements the Combiner role. Arguments: 1. txs (json array, required) The base64 strings of partially signed transactions [ "psbt", (string) A base64 string of a PSBT ... ] Result: "str" (string) The base64-encoded partially signed transaction Examples: > elements-cli combinepsbt '["mybase64_1", "mybase64_2", "mybase64_3"]'
